<h1>Mooview</h1>
 <p>mooview is a web application about movies and TV shows, inspired by the <a href='https://www.themoviedb.org/?language=id-ID'>TMDB</a> website.
  all movie and tv show data used in this app is provided by <a href='https://developer.themoviedb.org/reference/intro/getting-started'>TMDB api</a>.</p>
  <h3>Frontend Tech</h3>
 <ul>
   <li>React</li>
   <li>Vite</li>
   <li>Chakra UI</li>
   <li>React Router</li>
   <li>React Slick</li>
 </ul>
 <h3>Backend Tech</h3>
 <ul>
   <li>Nest JS</li>
   <li>Mysql</li>
   <li>Prisma ORM</li>
 </ul>
